Welcome to the **Sovereign Engine** ecosystem. To establish a secure, deterministic link between your infrastructure and the S.L.A.V.K.O.™ Kernel, follow this "Bunker Initialization" protocol.

## Step 1: Initialize the Bunker

Deploy the core backend services and establish the **Directive Zero** validation layer.

<AccordionGroup>
  <Accordion icon="clone" title="Clone the Core Repository">
    ```bash
    git clone https://github.com/mladengertner/sovereign-engine-core.git
    cd sovereign-engine-core
    ```
  </Accordion>

  <Accordion icon="box-open" title="Install Deterministic Dependencies">
    The Sovereign Engine uses strict, pinned dependencies to ensure 100% architectural fidelity.
    ```bash
    npm install
    ```
  </Accordion>
</AccordionGroup>

## Step 2: Establish Cryptographic Anchoring

Configure your **AuditChain** and verify the "Zero Temperature" logic environment.

<AccordionGroup>
  <Accordion icon="key" title="Configure Environment">
    1. Copy the example configuration: `cp .env.example .env`
    2. Define your `AUDITCHAIN_SECRET` and `SOVEREIGN_NODE_ID`.
    3. Ensure your local `Ollama` or `NVIDIA NIM` instance is reachable.
  </Accordion>

  <Accordion icon="shield-check" title="Perform Deterministic Type Check">
    Verify that your logic adheres to the absolute syntactical perfection required by the **Sovereign WAF**.
    ```bash
    npm run typecheck
    ```
  </Accordion>
</AccordionGroup>

## Step 3: Ignite the Engine

Activate the backend orchestrator and initialize the Visual Theatre.

<Accordion icon="bolt" title="Launch the Backend">
  ```bash
  npm run backend:dev
  ```
  *Observe the terminal for the "SOVEREIGN STATUS: YELLOW" signal, indicating the system is healthy and ready for kognitive orchestration.*
</Accordion>
